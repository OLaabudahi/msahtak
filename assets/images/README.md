# Assets Required

Please add the following images to this folder:

1. **msahtak_logo.png** - The Msahtak logo (as shown in the UI design)
   - Recommended size: 800x400 px or similar ratio
   - Format: PNG with transparent background

2. **google_icon.png** - Google sign-in icon
   - Recommended size: 96x96 px
   - Format: PNG

3. **apple_icon.png** - Apple sign-in icon
   - Recommended size: 96x96 px
   - Format: PNG

You can extract these images from your UI design files or use similar icons from icon libraries.
