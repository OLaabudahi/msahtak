# Msahtak App

A Flutter application for the Msahtak platform.

## Features

- User authentication (Sign up, Login)
- Social sign-in (Google, Apple)
- Notifications system
- Multi-language support (English, Arabic)

## Setup

```bash
flutter pub get
flutter run
```

## Requirements

- Flutter SDK 3.10.4+
- Firebase configuration
